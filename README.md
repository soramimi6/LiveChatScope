# LiveChatScope

YouTube ライブ配信のチャットリプレイを取得・分析し、任意の再生時刻へジャンプできる Web ツール。

## 概要

- **対象**: ライブチャット（コメント欄ではない）
- **主用途**: 配信終了後、URL からチャットリプレイを一括取得（Phase A）
- **将来**: 安定稼働後に複数ユーザー向け公開サービスを検討（Phase D）

## 主な機能（予定）

1. URL 入力によるチャットリプレイ取得
2. 検索・フィルタ・時系列分析
3. コメント密度・スパイク検出・投稿者ランキング等
4. `watch?v=...&t=...s` 形式での任意時刻ジャンプリンク生成
5. JSON / CSV エクスポート
