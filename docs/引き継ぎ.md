## 4. コード構成（実装後）

### Backend（`backend/`）

```
app/
  main.py              # FastAPI + CORS + lifespan DB init
  api/
    videos.py          # POST/GET videos, status, display_filter
    analysis.py        # summary, topics, keywords/bursts, authors/{id}/profile, refilter
    messages_api.py    # GET /messages（FTS5）
    export.py          # GET /export/{type}（JSON v2, フィルター対応）
    export_names.py    # UX-21: LiveChatScope_Result_* ファイル名
    common.py          # jump_url, time_text, 409 制御
  services/
    fetch_worker.py    # chat-downloader 取得 + video_metadata
    author_profile.py  # C1: 常連プロフィール集約
    analysis/          # pipeline, refilter_pipeline, message_filter, stage0–8, stage4b
db/schema.sql
config/analysis_defaults.json
tests/                 # 54 passed（2026-06-21）: test_keyword_bursts, test_author_profile, ...
```

### Frontend（`frontend/`）

```
app/                   # /, /analyze/[id], /videos/[id]
components/
  global-filter-bar.tsx  # UX-24/19 フィルター + セッション NG
  keyword-burst-ranking.tsx   # B1: 急上昇キーワード
  topic-super-chat-ranking.tsx # D2: 話題別 SC
  author-profile-sheet.tsx    # C1: 常連プロフィール Sheet
  ui/sheet.tsx         # shadcn Sheet
  tabs/                  # summary, topics, highlights, revenue, community, search
  export-menu.tsx
  jump-link-button.tsx, kpi-card.tsx, topic-timeline-bar.tsx, ...
lib/
  api.ts               # videos, summary, display_filter, refilter, keywords/bursts, author profile
  topic-super-chat.ts  # D2: 話題別 SC 集計ヘルパ
  session-filter.ts    # UX-19 セッション NG（sessionStorage）
  api/highlights.ts, revenue.ts, community.ts, search.ts, export.ts
  export-filename.ts   # UX-21: クライアント側 DL 名
  mocks/               # summary, topics, highlights, revenue, ...
```

## 7. 開発環境

### 7.1 必要ソフト

| ソフト | 要件 |
|--------|------|
| Python | 3.11+（Windows Store スタブ不可） |
| Node.js | LTS 推奨 |
| Git | — |

### 7.2 Backend セットアップ

```powershell
cd backend
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
pip install -r requirements-dev.txt   # pytest, httpx（E2E 用）
uvicorn app.main:app --reload --port 8000
```

- DB: `data/livechatscope.db`（gitignore。初回起動で自動作成）
- エクスポート: `data/exports/{video_id}/`（Stage 8）

### 7.3 Frontend セットアップ

```powershell
cd frontend
copy .env.example .env.local    # NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
npm install
npm run dev
```

- http://localhost:3000
- `npm run build` — 成功確認済み

### 7.4 `.venv` / `node_modules`

**git に含まれない**。別 PC では上記コマンドで再生成。
