# LiveChatScope — プロダクト概要

## 名称

**LiveChatScope** — ライブチャットを「覗く・分析する」スコープ（観測装置）のイメージ。

## 背景

YouTube ライブ配信のチャットリプレイは、配信者・分析者にとって重要なデータだが、
終了後に公式 API だけで全件取得することはできない。
LiveChatScope は取得から分析・時刻ジャンプまでを一つの Web ツールで提供する。

## フェーズ

### Phase A（MVP）

- 配信終了後、YouTube URL を入力してチャットリプレイを取得
- 検索・基本分析・ジャンプリンク・エクスポート
- 個人利用・動作検証

### Phase B（本番品質）

- ジョブ進捗・中断再開
- エラーハンドリング強化
- 大規模配信（10万コメント級）の性能確認

### Phase D（公開サービス・将来）

- ユーザー認証・レート制限
- 同一動画のキャッシュ共有
- 利用規約・プライバシー対応

## 技術方針（概要）

- **取得**: chat-downloader（Python）でリプレイチャットを取得
- **分析**: `time_in_seconds` を軸に時系列集計・検索
- **ジャンプ**: `https://www.youtube.com/watch?v={VIDEO_ID}&t={秒}s`
- **Web**: フロント + API サーバー + DB（詳細は architecture.md）

## 制約・リスク

- 配信者がチャットリプレイを無効にしている場合は取得不可
- 非公式取得手段に依存するため、YouTube 仕様変更で障害の可能性あり
- 公開サービス化時は ToS・プライバシー・ストレージコストの検討が必要
