# LiveChatScope — プロダクト概要

## 名称・定位

**LiveChatScope** — 配信後のチャットを、**個人配信者の振り返り資料**に変える Web ツール。

YouTube Analytics の代替ではなく、**チャットリプレイに基づく配信運営の振り返り**に特化する。

## ターゲット

- **主**: 個人配信者（盛り上がり把握・スーパーチャットお礼・切り抜き候補）
- **副**: 協力者（編集者・モデレーター）— エクスポート・ジャンプリンク共有

## 提供価値

| 課題 | 答え |
|------|------|
| どこが盛り上がった？ | 盛り上がり候補 + ジャンプ URL |
| 配信の構成は？ | 話題ブロック・構成タイムライン |
| スーパーチャットのお礼 | お礼リスト（時刻・金額・ジャンプ） |
| 切り抜きどこから？ | 切り抜き候補パック |
| 間延びした区間は？ | 低活動区間の可視化 |

## フェーズ

| Phase | 内容 | 状態 |
|-------|------|:----:|
| **A** | 取得・基本分析・検索・ジャンプ・エクスポート | ✅ |
| **A+** | サマリー・盛り上がり・話題・キーワード・配信者向け出力 | ✅ |
| **B** | 中断再開・大規模性能（10 万コメント級） | 📋 |
| **D** | 認証・公開サービス | 将来 |

## 機能要件（第一弾）

### 取得・閲覧

- YouTube URL 入力 → チャットリプレイ非同期取得 → 進捗表示
- メッセージ一覧・キーワード検索・投稿者フィルタ
- ジャンプ URL（`watch?v=&t=Ns`）

### 分析 — 基本

- コメント密度タイムライン、Top 投稿者、メッセージ種別集計、スーパーチャット集計

### 分析 — 差別化（A+）

| 機能 | 内容 |
|------|------|
| 盛り上がり候補 | 密度スパイク Top N + ジャンプ |
| キーワード | 形態素解析 + ストップワード |
| 話題ブロック | 語分布変化点で区間分割・ラベル |
| 話題遷移 | ブロック間の切り替わり |
| 話題別 Top 投稿者 | ブロックごとのランキング |
| 低活動区間 | 密度が低い連続区間 |
| 常連コア層 | 複数ブロックに参加する投稿者 |
| 配信サマリー | KPI + 構成 TL + preview |

### エクスポート

- JSON / CSV / Markdown（切り抜き・お礼・サマリー）

## スコープ外

- 通常コメント欄、配信中リアルタイム、動画プレイヤー埋め込み
- 同時視聴者数等（Analytics 領域）
- 英語 UI（第一弾）

## 制約

- チャットリプレイ無効の配信は取得不可
- chat-downloader 依存（YouTube 仕様変更リスク）
- 話題ラベルは「チャット上の推定話題」
