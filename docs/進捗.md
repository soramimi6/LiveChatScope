#### #5 Batch D 仕様確定・実装完了（2026-06-21）

> 以下が確定仕様（ユーザー承認済み）。**2026-06-21 実装完了**。

##### 方針

- **分析目的** — エンタメ性は求めない。グラフが寂しくても（0〜2 件等）**許容**。
- **Subscribe** — データ源にないため UI・API・集計 **一切触れない**。

##### データ源（#4 取得済み）

| message_type | 意味 | `author` の指す相手 |
|--------------|------|---------------------|
| `membership_item` | メンバーシップ登録 | **登録した視聴者** |
| `sponsorships_gift_purchase_announcement` | ギフト購入告知 | **ギフトを贈った視聴者** |
| `ticker_sponsor_item` | ticker 重複 | **取り込まない** |

##### 共通ルール

| 項目 | 仕様 |
|------|------|
| **重複・再登録** | 同一 `author_id` は **1 回としてカウント**（リスト・KPI）。グラフのバケット件数は **ユニーク author 数** |
| **負の時刻** | 配信開始前の可能性が高い → **`0:00:00`（0 秒）として扱う** |
| **時刻欠損** | グラフから除外。リストは「時刻不明」行として残す |
| **「配信中に登録」** | リプレイ内 `membership_item` ＝ 配信中登録（観測可能な範囲の定義） |

##### メンバーシップ登録

| 要件 | 仕様 |
|------|------|
| 時刻 | 各登録イベントの `time_in_seconds`（負値は 0 秒に正規化） |
| クラスター | 60 秒バケット → 移動平均 baseline 比 **burst_score**（`keyword_bursts` / `highlights` 同系） |
| グラフ | 登録件数タイムライン + burst ピークハイライト。**件数少でも表示**（空状態 UI は最小限） |
| **ピーク近接コンテキスト（v1 含む）** | burst ピーク付近に **話題ブロックラベル**・**盛り上がり候補** を時刻近接で表示（例: 「01:04:00 付近 — 話題: ガンプラ / 盛り上がり #2」） |
| ユーザーリスト | **「配信中に登録」** バッジ。表示名・author_id・登録時刻・jump リンク。author あたり 1 行 |

##### ギフト告知

| 要件 | 仕様 |
|------|------|
| グラフ | **不要** |
| マーキング | **「ギフト告知を使用」** バッジ |
| リスト | 告知ユーザー一覧（時刻・jump）。author 重複は **1 回カウント** |
| KPI | 件数のみ（コミュニティタブ上部） |

**技術メモ（スーパーチャット — 2026-06-21 調査）**:

| 項目 | 内容 |
|------|------|
| 症状 | DB 上 `uP51LV8d7mw`（53,690 件）は **`text_message` のみ**。`super_chat_*` テーブルも空。収益タブは `none_in_chat` |
| 原因① | `fetch_worker.py` が `ChatDownloader().get_chat(url)` を **引数なし**で呼んでいる。chat-downloader のデフォルト `message_groups=['messages']` は **`text_message` のみ**取得。`paid_message` / `paid_sticker` 等（superchat・tickers グループ）は**最初から除外**される |

| 原因③ | ticker 系・メンバー系は `time_in_seconds` が負になることがあり、Stage0 の `skip_negative_time` で**削除される**可能性 |
| 検証 | テスト動画 `UlemRwXYWHg` で `message_groups=['messages']` → SC 0 件、`['messages','superchat','tickers']` → `paid_message` 等を取得・`money.amount` もパース OK |

## 4. ブランチ・作業ルール

```
master  … リリース
  ↑

  ↑
  cursor/* / feat/* / fix/* … 作業ブランチ
```
