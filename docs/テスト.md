# LiveChatScope — テスト

## E2E 自動テスト（推奨）

```bash
./scripts/e2e-api.sh                                    # スモーク
./scripts/e2e-api.sh "https://www.youtube.com/watch?v=VIDEO_ID"  # フルフロー
```

Windows: `.\scripts\e2e-api.ps1 -Url "..."`

フルフローは POST → ポーリング → 分析 API → エクスポートを自動検証（タイムアウト 30 分）。

## E2E-01 手動チェック（任意）

1. URL 入力 → 進捗画面
2. 取得・分析完了を待つ
3. サマリー: KPI・話題 preview
4. 盛り上がり: ジャンプ URL が YouTube で該当秒付近から再生
5. 収益: スーパーチャット表示 or Empty
6. 詳細検索: キーワード 1 件以上ヒット
7. JSON / CSV エクスポート成功
8. 切り抜き / お礼 Markdown 取得

サンプル URL はリポジトリに固定しない。チャットリプレイ有効・配信終了済み・1,000 件以上を都度選定。

## 既知の制限

| 制限 | 説明 |
|------|------|
| 非公式取得 | [Indigo128/chat-downloader](https://github.com/Indigo128/chat-downloader) fork 固定 |
| 推定話題 | 配信内容とチャット話題は一致しない場合がある |
| 認証なし POC | 公開ネットワークへの無防備デプロイ不可 |
| 英語 UI | 第一弾対象外 |
