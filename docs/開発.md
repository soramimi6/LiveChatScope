# LiveChatScope — 開発

## ブランチ

| ブランチ | 用途 |
|----------|------|
| `master` | リリース相当 |
| `dev` | 日常の統合ブランチ |
| `feat/*`, `fix/*`, `docs/*` | 作業ブランチ（`dev` から分岐） |

作業ブランチは `dev` から切り、完了後 `dev` にマージする。

## タスク実行

- 実装タスクは **メインチャット → Task サブエージェント** で実行
- グローバルルール: Cursor `multi-agent-development.mdc`
- 判断に迷ったら盛る → 後で削る。UI は日本語のみ（第一弾）

## 環境構築

### 必要ソフト

Python 3.11+、Node.js、Git

### Backend

```bash
cd backend
python -m venv .venv
source .venv/bin/activate          # Windows: .\.venv\Scripts\Activate.ps1
pip install -r requirements.txt -r requirements-dev.txt
uvicorn app.main:app --reload --port 8000
```

- DB: `data/livechatscope.db`（初回起動で自動作成）
- エクスポート: `data/exports/{video_id}/`

### Frontend

```bash
cd frontend
cp .env.example .env.local         # NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
npm install
npm run dev
```

`.venv` / `node_modules` は git 管理外。clone 後に再生成する。
